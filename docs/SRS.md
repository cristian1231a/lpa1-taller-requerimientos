# Lista de requerimientos
# R1: El sistema debe registrar un nuevo hotel con información básica, nombre, dirección, teléfono, correo electrónico y ubicación.
# R2: El sistema debe registrar los servicios del hotel, como restaurante, piscina, gimnasio.
# R3: El sistema debe registrar imágenes.
# R4: El sistema debe registrar 
# R5: El sistema debe registrar las habitaciones, el cual permita un registro de su tipo, descripción, precio, servicios incluidos, capacidad y registro de fotos.
# R6: El sistema debe registrar condiciones de pago, con múltiple opción ya que cada hotel maneja diferente condición de pago, puede ser pago completo por adelantado o pago al llegar al hotel.
# R7: El sistema debe registrar las políticas de cancelación, dependiendo del tipo de habitación, temporada y del hotel. Con opción de escoger una fecha límite de cancelación y de cobrar un valor o porcentaje de penalidad.
# R8: El sistema debe registrar diferentes estados del hotel registrado. Con diferentes opciones de estado, como activo, en mantenimiento, remodelación o cerrado.
# R9: El sistema debe registrar diferentes estados de las habitaciones registrados, las múltiples opciones disponible, reservada, en mantenimiento, remodelación o desinfección, y mientras estén en una opción diferente a disponible no podrá ser reservada.
# R10: El sistema debe registrar un calendario de temporadas con rango de fechas para temporadas altas y bajas.
# R11: El sistema debe registrar dentro del calendario dependiendo de la temporada, un valor porcentaje para que se actualice el precio base del valor de la habitación. Y también registrar el valor de las promociones de cada temporada.
# R12: El sistema debe registrar fecha para cada habitación, donde se pueda guardar que días están reservadas y así mismo saber cuándo están disponibles.
# R13: El sistema debe registrar las calificaciones del huésped después de su estancia, esta calificación puede estar dividida en dos contenedores y en estrellas, calificación de la habitación, calificación del hotel y una caja de comentarios.
# R14: El sistema debe registrar el promedio de estas calificaciones y publicarlo en cada habitación individual.
# R15: El sistema debe registrar un nuevo cliente, donde se le solicite, nombre completo, numero de teléfono, correo y dirección.
# R16: El sistema debe registrarlos criterios de búsqueda del cliente, podrá buscar de acuerdo a las fechas que necesita, ubicación y precio.
# R17: El sistema debe mostrar esta búsqueda realizada por el cliente, donde aparecerán múltiples opciones de acuerdo a su búsqueda.
# R18: El sistema debe permitir que el cliente ingrese a ver los detalles de la habitación que le interese.
# R19: El sistema debe permitir ver al cliente los servicios, fotos, calificación ofrecidos de la habitación y del hotel, 
# R20: El sistema debe permitir al cliente seleccionar la habitación, y proceder a pagar.
# R21: El sistema debe registrar el pago, las fechas, la habitación escogida del cliente.
# R22: El sistema debe automáticamente cambiar el estado a reservada de la habitación cuando el cliente haya formalizado la reserva.
